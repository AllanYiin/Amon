# SVG → PPTX 輸出流程（製作階段參考）

當使用者要求把已完成的 SVG 投影片輸出成 PPTX 時使用。

## 前置

- 安裝 `svg_path-7.0-py2.py3-none-any.whl`
- 使用專案檔案 `svg2pptx.py` 的工具函數

## 規則

- 必須以「SVG 原尺寸比例」建立對應寬高比的 PPTX。
- 必須依頁碼順序（slide1.svg, slide2.svg… 或清楚的排序規則）置入投影片。
- 嚴禁以「先轉 PNG 再貼到投影片」作為替代方案。

## 交付

- 最終交付應包含：
  - 定稿 PPTX
  - 定稿 SVG（全套）
  - 對應圖檔（如 PNG 預覽）
- 將以上內容打包壓縮後提供下載連結。
