# 視覺設計規則（製作階段參考）

本文件在你要把「內容規劃」落地成 SVG/PPTX 投影片時使用。

## 1) 色彩與風格

- 禁止使用：漸層、陰影、框線、圓角、描邊（含 stroke）
- 避免半透明文字/圖形（會降低可讀性）
- 預設主色調：橙色 + 灰色；可依需求挑選少量輔色，但整份不宜過雜
- 圖層順序：箭頭／連接線／色塊需壓在最底層（避免壓住文字或圖示）

## 2) 文字規則

- 投影片上除資料來源與註解外，不得使用小於 12pt 文字
- 強調建議以「文字顏色變化」凸顯；整份不超過 1–2 種主要字體顏色
- 禁用 stroke 屬性（會造成文字模糊）

## 3) 版面與動線

- 畫面比例預設 16:9
- 視覺動線以三角形「底邊 → 尖端方向」為準（常見錯誤為反向）
- 避免視覺區塊過於單調：可用色塊明度/透明度差異拉出層次（差異至少 25%）

## 4) 視覺化元素要求

- 禁止僅文字無視覺化的頁面
- 圖像可包含：icon、線條、色塊、圖表、表格、infographic 模組
- 涉及新聞、數據時：同頁下方需用小字註明來源（≥12pt 規則例外：僅此處可較小，但仍需可讀）

## 5) 風格建議

- 若使用者未特別指定：以「瑞士國際字體風格」或「Bento 設計風格」作為設計精神
- 封面頁是關鍵：必須有主視覺與清楚層次，不能只有素面文字
