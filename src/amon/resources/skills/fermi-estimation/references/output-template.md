# Fermi Estimation Output Template

預設輸出順序如下，除非使用者明確要求更短版本。

## 1. 適用性判斷
- 這題是否適合費米估計
- 若不適合，直接說原因並停止

## 2. 問題定義
- 估算目標
- 單位
- 地理範圍
- 時間範圍

## 3. 因子拆解
- 用 3-7 個因子把目標值拆開
- 清楚寫出每個因子和目標值的關係

## 4. 假設表
- 因子名稱
- 最可能值
- 樂觀估計
- 悲觀估計
- 估值理由
- 說明樂觀/悲觀在這題分別對結果代表什麼方向，不可省略

## 5. 計算過程
- 公式
- 代入數值
- 最可能值計算
- 樂觀情境計算
- 悲觀情境計算

## 6. 結果
- 最可能值
- 由樂觀/悲觀情境形成的上下界
- 若題目是合理性檢查，直接回答「合理 / 偏高 / 偏低」

## 7. 驗證
- 至少一個 sanity check
- 說明哪個假設最影響結果

## 8. 侷限
- 哪些是假設
- 哪些因素未納入
- 什麼情況下需要改用精確查證
- 若樂觀/悲觀方向與一般直覺相反，需明講原因
