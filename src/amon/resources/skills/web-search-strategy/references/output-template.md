# Output Template

預設輸出順序如下，必要時可縮短，但不要少掉「未解空缺」與「下一輪建議」。

## 研究目標
- 使用者要找什麼
- 時效性或地區限制
- 來源偏好

## 查詢策略
用表格或條列列出：
- 查詢內容
- 為何這樣查
- 預期打到的頁面類型
- 是否有限定網站、檔案、日期或語言

## 主要發現
- 只寫已查到的內容
- 對每個重要結論附來源類型或連結

## 未解空缺
- 還缺哪些證據
- 哪些地方只有二手來源，尚未找到原始來源
- 哪些是推論，不是已驗證事實

## 下一輪建議查詢
- 若需要第二輪搜尋，列出 1-3 組更精準查詢
- 說明每組是為了補哪個缺口

## Sources
- 只列真正使用過的來源
- 優先官方或原始來源
