# Responsive & Typography 指南

## Responsive Design
- 以 mobile-first 設計，逐步加強 tablet/desktop。
- 主要斷點：375 / 768 / 1024 / 1280px。
- 內容版面在小尺寸優先「單欄 + 主要 CTA」。

## Typography
- 文字是頁面核心，優先建立清晰階層。
- 預設 line-height ≥ 1.5，提升可讀性。
- 使用 `clamp()` 建立流體字級，避免在不同尺寸跳動過大。
